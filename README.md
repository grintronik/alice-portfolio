# Портфолио Алисы Домрачевой

Статический mobile-first сайт без сборщика и зависимостей.

## Локальный просмотр

```bash
python3 -m http.server 4173
```

Открыть `http://localhost:4173`.

## PDF

Положить готовый файл по адресу `assets/about/portfolio.pdf`. Кнопка скачивания появится автоматически.

## Подписи

Временные названия и годы находятся в начале файла `app.js`.
